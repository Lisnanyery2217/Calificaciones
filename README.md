
El ejercicio consiste en crear un formulario que permita calcular las 4 notas de un estudiante y medir mediante el resultado si el estudiante paso a reprueba


Descripcion:

el programa calcula las 4 notas de un estudiante para sacar su promedio y decir di aprobo , si este es insuficiente el programa habilita otra casilla la cual sirve para ingresar una nota extra , en este caso la nota del completivo , si aun así el promedio sigue siendo insuficiente el programa habilita nuevamente otra casilla la cual sirve para ingresar otra nota extra(el extraordinario), ya por ultimo el programa en base a las notas q le ingresaste anteriormente te muestra el promedio del estudiante y el estado del mismo.

Explicacion de como usar el programa:

Ingresa tus notas donde dice nota 1,2,3,4 Cuando hayas ingresado todas las notas, haz clic en el botón Calcular.El sistema hará lo siguiente automáticamente:
Calculará tu promedio, Mostrará el resultado en el campo Promedio.  Si el promedio es mayor a 69, el sistema muestra “Aprobado” y no habilita otros campos.
Si el promedio es 69 o menor, se habilita Completivo.
Si con el completivo llegas a 70 o más, quedas Aprobado y no usas extraordinario.
